Function

ILPBs is a web-based database of plant and biological stress interactions. It contains the following features:

  • Differentially expressed gene set analysis and search

  • Transcription factor(TF) analysis and target gene networks

  • Expression and query of resistance gene(R gene)

  • Construction and presentation of co-expression networks
